I am a Cyber Security Engineer, Security Researcher, Automation Developer, and Technical Writer with a deep passion for building secure systems, solving complex security challenges, and creating practical cybersecurity solutions.

My work spans multiple domains, including Red Teaming, Blue Teaming, Digital Forensics, Malware Analysis, Cloud Security, DevSecOps, Security Automation, Threat Hunting, Infrastructure Security, and Capture The Flag (CTF) development.

I take great satisfaction in designing environments that simulate real-world attack scenarios while also developing defensive mechanisms that can detect, prevent, and respond to sophisticated threats. For me, cybersecurity is not just a profession. It is a continuous journey of learning, experimenting, researching, and sharing knowledge. I firmly believe that understanding how systems fail is the first step toward building systems that are truly resilient.

Over the years, I have worked on a wide variety of cybersecurity projects involving enterprise infrastructure, cloud environments, automation pipelines, security monitoring, and offensive security research.
